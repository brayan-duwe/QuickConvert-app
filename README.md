# QuickConvert 🔃

This is a very simple app that converts measurement units, such as Meters, Kilometers, Miles, Yards and Feet.

## Main files of the project
|Function| File |
| -------|-------- |
|View & UI | [ContentView.swift](/QuickConvert/ContentView.swift) |
| Functions to convert units | [UnitCoverter.swift](/QuickConvert/UnitConverter.swift) |

The app is divided into 3 sections:
1. **Types of conversion.** A picker indicating the unit **From**, and a picker indicating **To**. Between them is a button, which swaps the units, very practical.
2. **Input.** A text field to insert the unit to be converted.
3. **Result.** It is determined by the function's return.
